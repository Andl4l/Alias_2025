# Alias 2025

Современная мобильная версия популярной словесной игры для компании. 

## Описание

Alias 2025 - это игра для развития коммуникативных навыков, расширения словарного запаса и веселого времяпрепровождения в кругу друзей или семьи.

### Основные возможности

- Поддержка русского и английского языков
- Неограниченное количество команд
- Три уровня сложности
- Настраиваемая длительность раунда
- Система штрафных баллов
- Интуитивное управление свайпами
- Современный дизайн

## Требования

- iOS 17.0+
- Xcode 15.0+
- Swift 5.0+

## Установка

1. Клонируйте репозиторий
```bash
git clone https://github.com/ваш-username/Alias-2025.git
```

2. Откройте `Alias 2025.xcodeproj` в Xcode

3. Запустите проект на симуляторе или реальном устройстве

## Использование

1. Создайте минимум две команды
2. Настройте параметры игры (язык, сложность, время)
3. Начните игру
4. Объясняйте слова своей команде
5. Свайпайте вверх для угаданных слов, вниз для пропущенных
